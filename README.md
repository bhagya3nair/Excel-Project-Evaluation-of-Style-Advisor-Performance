Bare International Analysis – Style Advisor Performance Evaluation

Project Overview:

This project focuses on analyzing the performance of Style Advisors in retail environments using evaluation data collected through store audits. The analysis examines multiple performance criteria such as store ambiance, first impressions, customer interaction, trial experience, and service quality to determine how effectively Style Advisors contribute to customer satisfaction and sales performance.

The objective of the project is to transform raw evaluation data into actionable insights that can help retail management improve service quality, optimize staffing, and enhance customer experiences. The project includes Excel-based analysis, pivot table reporting, performance grading, and visualization of key metrics.

Business Problem:

Retail organizations rely heavily on Style Advisors to deliver personalized shopping experiences and influence purchasing decisions. However, inconsistent performance across regions or advisors can negatively affect customer satisfaction and revenue.

This project aims to answer key questions such as:

Which regions demonstrate the strongest advisor performance?
What percentage of advisors are high performers?
Are there trends in performance over time?
What factors influence evaluation scores?

By analyzing these aspects, businesses can identify improvement opportunities and implement targeted training strategies.

Objectives of the Analysis:

The project focuses on the following analytical goals:

Assign performance grades to Style Advisors based on evaluation scores.
Analyze average evaluation scores across different regions.
Identify the region with the highest percentage of high performers.
Use data visualization techniques to highlight performance trends.
Provide recommendations for improving advisor performance.

These objectives provide a structured framework for evaluating operational performance within retail environments.

Dataset Overview:

The dataset contains evaluation records collected during store visits and includes multiple performance indicators.

Key Data Fields:

Evaluation_ID – Unique identifier for each evaluation record
Evaluation_Date – Date of evaluation
Evaluation_Score – Overall performance score
Store Ambiance Metrics – Signage quality, lighting, and store presentation
Customer Interaction Metrics – Discovery, communication, trial experience
Location Data – City, state, zone, and country
Visit Details – Entry time, exit time, and visit duration
Auditor Demographics – Age, gender, attire
Advisor Information – Style advisor name and description

These variables allow for multi-dimensional analysis of performance across geography, staffing, and operational factors.

Analytical Approach:

The project applied several Excel data analysis techniques to derive insights.

1. Performance Grading

Style Advisors were categorized into four performance levels:

Category	Score Range
High Performer	90 – 100
Average Performer	70 – 89
Low Performer	50 – 69
Bottom Performer	Below 50

Performance distribution from the analysis:

High Performers: 16%
Average Performers: 44%
Low Performers: 26%
Bottom Performers: 13%

This grading system helps quickly identify top-performing advisors and those requiring improvement.

Pivot Table Analysis:

Pivot tables were used to analyze average evaluation scores by zone.

Key Observations:

North Zone: Highest average evaluation score (77)
South Zone: Lowest average evaluation score (67)
Overall Average Score: 72

These results highlight regional performance differences and help identify areas requiring operational improvement.

High Performer Analysis:

A separate pivot analysis was conducted to determine the percentage of high-performing advisors by region.

Key Finding:

North Region has the highest percentage of high performers (35%)

This suggests that the North region demonstrates stronger operational practices, staff training, or customer engagement strategies.

Visualizations Created:

Several charts were used to make insights easier to interpret:

Bar Chart: Average evaluation scores by zone
Pie Chart: Percentage distribution of high performers by region
Line Chart: Performance trends over time
Heatmap: Evaluation score breakdown by criteria
Stacked Bar Chart: Comparison of high vs low performers

These visualizations help stakeholders quickly understand performance patterns and identify improvement opportunities.

Key Insights:

Regional Performance

North region leads in evaluation scores and high performer percentage.
South region shows lower performance levels and may require targeted improvement strategies.

Performance Distribution

A majority of advisors fall within the average performance category (44%), indicating moderate performance overall.

Staffing Impact

Locations with 3–4 Style Advisors showed more consistent performance, suggesting adequate staffing improves customer experience.

Auditor Demographics

74% of auditors were male and 26% female.
Female auditors tended to give higher evaluation scores.

Visit Duration

Most customer visits lasted between 500–2500 seconds.
Visit duration had minimal impact on evaluation scores, suggesting service quality matters more than time spent.

Performance Gap

Top performers achieved scores above 90, demonstrating strong service quality.
Bottom performers averaged around 35, indicating a significant improvement opportunity.

Business Recommendations

Based on the analysis, several actions are recommended:

Improve Training Programs

Implement targeted training programs focusing on:

Customer engagement
Product knowledge
Upselling techniques
Communication skills
Optimize Staffing Levels

Stores with more Style Advisors demonstrated more consistent service performance.

Performance Monitoring:

Establish regular evaluation reviews and feedback sessions to continuously improve advisor performance.

Regional Knowledge Sharing:

Adopt best practices from the North region, which demonstrated the strongest performance results.
